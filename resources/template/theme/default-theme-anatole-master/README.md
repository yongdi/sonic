## 安装方法

1. [点击](https://github.com/go-sonic/theme-anatole/releases)选择最新的版本下载。
2. 进入后台 -> 外观 -> 主题。
3. 点击右下方按钮选择安装主题，随后选择 `本地上传`。
4. 选择下载好的主题包（zip）即可。
